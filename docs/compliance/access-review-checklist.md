# Quarterly Access Review Checklist

## Checklist

- [ ] Export current user and admin access lists from all systems (Azure, DB, logging, etc.)
- [ ] Review joiner–mover–leaver events for the quarter
- [ ] Validate least-privilege assignments for all roles
- [ ] Remove or justify any privileged or orphaned accounts
- [ ] Review and document all third-party/vendor access
- [ ] Capture evidence (screenshots, exports, approvals)
- [ ] Document review date, participants, and findings

## Evidence Template

| Date       | Reviewer   | System     | Finding/Action        | Evidence Link/Attachment |
| ---------- | ---------- | ---------- | --------------------- | ------------------------ |
| YYYY-MM-DD | Jane Doe   | Azure AD   | Removed stale account | screenshot.png           |
| YYYY-MM-DD | John Smith | PostgreSQL | All roles reviewed    | export.csv               |

## Next Steps

- [ ] Schedule next quarterly review
- [ ] File evidence in compliance repository
- [ ] Update access review log

_This checklist and template should be used for every quarterly access review and retained for audit evidence._

# Breach Notification Playbook

## 1. Initial Response

- [ ] Confirm breach and scope (PHI/PII involved?)
- [ ] Notify incident response team and management
- [ ] Contain and mitigate further exposure

## 2. Notification Timeline

- [ ] HIPAA: Notify affected individuals and HHS within 60 days
- [ ] GDPR: Notify authorities within 72 hours
- [ ] Document all notifications and evidence

## 3. Communication Templates

- [ ] Internal notification template
- [ ] External notification template (users, partners)
- [ ] Regulator notification template

## 4. Evidence & Documentation

- [ ] Collect and preserve all relevant logs and communications
- [ ] Document timeline and actions taken
- [ ] File evidence in compliance repository

## 5. Post-Breach Review

- [ ] Conduct post-mortem and update playbook
- [ ] Review and improve controls
